# Contributing to AgentsHive

Thanks for your interest! 🜁

## How to Contribute

### Report Bugs
Open an issue with:
- What you expected
- What happened
- Steps to reproduce

### Submit Code
1. Fork the repo
2. Create a feature branch
3. Submit a PR

### Add to the Cache
The best contribution is **using the mesh**. Every query you contribute makes the cache smarter for everyone.

### Build a Node
BCP v0.1 is open — build your own cache node that speaks the protocol. See [PROTOCOL.md](docs/PROTOCOL.md).

## Code Style
- Python 3.10+
- Type hints preferred
- No external API dependencies for core features
